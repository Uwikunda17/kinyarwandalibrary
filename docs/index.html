ikinyarwanda-lang

Build scripts in Kinyarwanda

Run Kinyarwanda commands in web apps, validate forms, call APIs, and execute .ikw scripts with a CLI.

Install

npm install ikinyarwanda-lang

Language

umukoro greet(name) {
  andika(name);
}

izina = 'Muraho!';
subiramo(i, 1, 3) {
  greet(izina);
  andika(i);
}

CLI

ikin run scripts/my-script.ikw

CLI only accepts files with the .ikw extension.
